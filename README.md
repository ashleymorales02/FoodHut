# Meal Generator (iOS)

iOS mobile application designed to solve "decision fatigue" by generating randomized meal options based on user preferences.

## 🌟 Highlights
- **Independently Built:** Developed while self-teaching the Swift programming language and Apple’s UIKit framework.

## 🛠 Tech Stack
- **Language:** Swift
- **Framework:** UIKit
- **IDE:** Xcode

## 📱 Features
- **Randomizer Engine:** Press a button to receive a meal suggestion from a curated or dynamic list.

## 🧠 Learning Journey
This project was a deep dive into the iOS ecosystem. Key challenges overcome include:
- Mastering Swift's type-safety and optionals.
- Understanding the ViewController lifecycle.
- Designing layouts that adapt to different iPhone screen sizes using Auto Layout.
